# AgentRecord
简易的Agent日记系统

每天的信息保存为一个独立的markdown文件

AI可以调用本地工具实现查阅、搜索、总结功能

严格的信息隔离机制，AI无法篡改原始记录流，只能修改总结部分

### 效果如下
```
╭────────────────╮
│ Agent 日记系统  │
╰────────────────╯
  可用模型: deepseek-v4-pro, gemini-3-flash-preview, gemini-3.1-pro-preview
╭────────────────────────────────── 命令手册 ──────────────────────────────────╮
│   /help            -> 显示此帮助                                             │
│   /model          -> 切换到下一个模型                                         │
│   /mode           -> 切换 记录/查阅 模式                                      │
│   /view [日期]     -> 查看历史日记（空=今天, /view help 查看所有用法）            │
│   /retry          -> 重试今日最后一个未回答的 @提问                             │
│   @[内容]          -> 呼叫AI解答或执行任务（如 @总结今日内容）                    │
╰──────────────────────────────────────────────────────────────────────────────╯

[deepseek-v4-pro | 记录] >> 
```
