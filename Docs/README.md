- [SQLite 数据模型](./SQLite数据模型.md)：表结构、状态机、版本链、事务边界、来源映射和升级规则。

阅读顺序建议：先看设计基线，再根据修改范围进入相应实现文档。代码与文档冲突时，应先判断代码是否偏离设计基线，而不是直接用现状覆盖基线。
